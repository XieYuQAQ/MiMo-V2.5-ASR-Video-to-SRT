💡 CORS 跨域解决方案

如果遇到跨域错误，可以使用以下任一方式解决：

方式一：使用 CORS 浏览器扩展

安装 Allow CORS 扩展，启用后刷新页面。

方式二：本地代理

运行本项目的 Flask 版本（python app.py），由服务端代理 API 请求。
